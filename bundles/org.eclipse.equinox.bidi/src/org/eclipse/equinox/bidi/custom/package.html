This package provides an interface and classes for developing structured text processors.
  • {@link ISTextProcessor} is an interface specifying which methods a processor must provide.
  • {@link STextProcessor} is a generic processor which can be used as superclass for specific processors.
  • {@link STextFeatures} allows managing some factors which affect a structured text processor's behavior
  • {@link STextStringProcessor} is a class which allows retrieval of the defined processor types and of the corresponding processors.
This package is to be used together with package {@link org.eclipse.equinox.bidi}.
